# Smart Career Guidance and Course Recommendation System

## Description
A web-based application that helps students choose suitable career paths based on their interests, skills, and preferences.

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Machine Learning (KNN)

## Features
- User Registration and Login
- Career Assessment
- Course Recommendation
- Skills Guidance
- College Recommendation
- Dashboard Interface
